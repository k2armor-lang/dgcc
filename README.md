# 디지털 게이트키퍼 (Digital Gatekeeper)

AI 시대의 진실 찾기 — 대한민국 고등학교 1~2학년 대상 미디어 리터러시 교육용 웹 시뮬레이션.
백엔드 없이 동작하는 **단일 HTML 파일**(React + Tailwind + Babel, 전부 CDN)입니다.

- 등장하는 인물·단체·언론사·학술지는 **전부 가상**입니다. (법령 조문/처벌 수위만 실제 현행법)
- 브라우저 저장소(localStorage 등)와 외부 서버 전송을 전혀 사용하지 않습니다.
- 딥페이크·가짜뉴스의 **판별법만** 다루며 제작법은 다루지 않습니다.

## 🌐 GitHub Pages로 배포하기

1. GitHub에서 새 저장소(repository)를 만듭니다. (예: `digital-gatekeeper`)
2. **이 폴더 안의 파일**(`index.html`, `README.md`)을 저장소에 업로드합니다.
   - 웹 업로드: 저장소 → **Add file → Upload files** → 드래그 → **Commit**
3. 저장소 **Settings → Pages** 로 이동합니다.
4. **Build and deployment → Source: Deploy from a branch**,
   Branch를 **`main` / `/(root)`** 로 선택하고 **Save**.
5. 1~2분 뒤 상단에 표시되는 주소(`https://<사용자명>.github.io/<저장소명>/`)로 접속하면 실행됩니다.

> `index.html`은 GitHub Pages가 자동으로 시작 페이지로 인식합니다. 별도 설정이 필요 없습니다.

## ✏️ 문제(콘텐츠) 교체 방법

`index.html`을 텍스트 편집기로 열어 **`const STAGES = [ ... ]` 배열의 문자열만** 수정하세요.
제목·본문·툴킷 단서·피드백·법령·강조 문구가 전부 문자열이며, 그 아래 코드는 건드릴 필요가 없습니다.

## 💡 온라인/오프라인 버전

- **이 폴더(`index.html`)**: CDN에서 라이브러리를 불러오는 **온라인 배포용**. 가볍고(약 80KB) 최신 라이브러리를 사용합니다. 실행에 인터넷 연결이 필요합니다.
- 인터넷이 불안정한 교실에서는 상위 폴더의 **`offline-classroom/index-offline.html`**(약 3MB, 네트워크 불필요)을 사용하세요.

## 👥 모둠 활동 · 판단 근거 · 교사 대시보드

- **모둠 모드**: 인트로에서 세션 코드(교사 안내)·모둠명·조원을 입력하고 모둠이 함께 진행합니다.
- **판단 근거**: 각 스테이지에서 REAL/FAKE를 고르기 전에 "왜 그렇게 판단했는지"를 반드시 작성합니다(10자 이상).
- **교사 대시보드**: 상단 **교사용** 버튼(또는 주소 뒤에 `#teacher`) → 모둠 결과를 한 페이지에 모아 보고, **스테이지별 판단 근거를 나란히 비교**(프로젝터로 다같이 보기)합니다.

### 실시간 취합을 켜려면 (선택)
`index.html` 상단의 `FIREBASE_CONFIG` 에 무료 Firebase 설정을 붙여넣으면, 모둠이 끝낼 때마다 교사 화면이 **실시간으로** 채워집니다. 설정법은 같은 폴더의 **`FIREBASE-SETUP.md`** 참고.

> 설정을 안 해도 됩니다. 그 경우 각 모둠 결과 화면의 **"결과 코드 복사"** → 교사 대시보드의 **"결과 코드 붙여넣기"** 로 취합합니다(외부 통신 없음).
>
> 개인정보 보호: 학생에게는 실명 대신 **모둠명**만 쓰게 하세요. Firebase를 켜면 입력 내용이 구글 클라우드로 전송됩니다.
